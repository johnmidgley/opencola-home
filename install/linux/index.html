Linux Installation

1. Download the Linux Release (if the download doesn't automatically start, click the link).

2. In a terminal, untar it wherever you like: 

> tar -xzf OpenCola-Linux-1.3.4.tgz

3. Start OpenCola 

> opencola/bin/opencola

4. A browser window will open with in app setup instructions

Note: During setup, you can choose to auto-start the application after reboots (assumes KDE).
